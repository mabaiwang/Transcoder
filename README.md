# Encode Studio

基于 C++ 和 Qt Widgets 的单窗口转码工具。

## 已实现功能

- URL 编码 / 解码
- HTML 实体编码 / 解码
- SQL_EN 编码 / 解码（`0x + UTF-16LE Hex`，与示例图一致）
- Hex 编码 / 解码
- Binary 编码 / 解码
- Asc（UTF-8 字节十进制）编码 / 解码
- Unicode Escape 编码 / 解码
- Base64 编码 / 解码
- MD5 32 位
- MD5 16 位
- SHA1
- SHA256

## 交互特点

- 所有编辑框默认一行高度，可通过“编辑框行数”统一调节
- 顶部输入一次，所有结果实时刷新
- 每张可逆卡片都支持单独粘贴内容后点击“还原”
- 每张卡片都支持一键复制
- 暗黑 / 浅色一键切换（默认浅色）
- 所有功能集中在一个窗口内
